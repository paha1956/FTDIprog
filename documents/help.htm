Инструкция по программированию целевого устройства.

 

1. Программатор FTDIprog предназначен для автоматизированного программирования цифровых бесконтактных амперметров, выполненных на основе микроконтроллера ATmega88. Также допускается программирование устройств на основе микроконтроллеров ATmega48 и ATmega168.

 

2. Подключите целевое устройство к порту USB персонального компьютера (ПК) с помощью кабеля FTDI USB - RS232.

 

3. Запустите программу FTDIprog.exe из папки FTDIprog.

 

4. Установите или снимите флаги "Проверка записанных данных" и "Защита FLASH памяти и EEPROM от копирования".

 

5. Нажмите клавишу "GO!" для начала программирования.

 

6. Дождитесь окончания программирования, признаком чего является вывод сообщения "Программирование устройства завершено!". Устройство считается нормально запрограммированным, если в конце каждого этапа программирования выводилась надпись "Ok".

 

Настройка параметров работы программы производится с помощью изменения соответствующих полей в файле конфигурации config.json. Формат файла config.json приведён ниже.

 

Примечания:

a. По умолчанию файл с данными для программирования FLASH памяти носит название flash_tc.hex и находится в папке с программой FTDIprog. Изменить расположение файла и его название можно, скорректировав параметр "FLASHfileName" в файле конфигурации config.json

 

b. Файл с данными для программирования памяти EEPROM по умолчанию носит название eeprom_tc_hex и находится в папке с программой FTDIprog. При отсутствии файла запись EEPROM производиться не будет. Изменить расположение файла и его название можно, скорректировав параметр "EEPROMfileName" в файле конфигурации config.json

 

c. Файлы flash_tc.hex и eeprom_tc.hex должны содержать данные в формате Intel HEX

 

 

 

Формат конфигурационного файла config.json.

 

Файл config.json содержит значения параметров работы программатора FTDIprog в формате JSON и должен располагаться в папке с программой FTDIprog.exe. Общий вид файла конфигурации приведён ниже:

 

{

    "FLASHfileName":  "flash_tc.hex",

    "EEPROMfileName": "eeprom_tc.hex",

    "LowFuseByte":    "EC",

    "HiFuseByte":     "DC",

    "ExtFuseByte":    "FF",

    "MemoryTest":     true,

    "MemoryLock":     false

}

 

Значения параметров файла конфигурации:

 

FLASHfileName содержит название файла прошивки FLASH памяти микроконтроллера в формате Intel HEX;

 

EEPROMfileName содержит название файла прошивки EEPROM памяти микроконтроллера в формате Intel HEX;

 

LowFuseByte, HiFuseByte и ExtFuseByte содержат значения младшего (low), старшего (hi) и дополнительного fuse байт соответственно в hex формате;

 

MemoryTest определяет значение флага проверки памяти после программирования по умолчанию при запуске программатора. Допустимые значения параметра: true - флаг установлен, проверка производится; false - флаг не установлен, проверка не производится;

 

MemoryLockt определяет значение флага защиты памяти микроконтроллера от копирования по умолчанию при запуске программатора.

Допустимые значения параметра:

true - флаг установлен, производится защита памяти;

false - флаг не установлен, защита не устанавливается.

 

Внимание! При изменении файла конфигурации необходимо сохранять расположение кавычек, двоеточий и запятых.
